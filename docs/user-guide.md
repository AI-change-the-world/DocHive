# 用户使用手册

## 快速入门

### 系统登录

1. 启动系统后，打开浏览器访问 `http://localhost:3000`
2. 输入用户名和密码登录系统
3. 首次使用可使用默认管理员账户登录

### 主界面介绍

系统主界面包含以下几个主要区域：

1. **顶部导航栏**: 包含系统Logo、用户信息和通知
2. **侧边栏菜单**: 提供系统功能导航
3. **主内容区**: 显示具体功能页面
4. **底部信息栏**: 显示系统版本等信息

## 文档管理

### 上传文档

1. 点击侧边栏"文档管理"菜单
2. 点击"上传文档"按钮
3. 选择本地文档文件
4. 系统自动解析文档内容并进行智能分类
5. 确认分类结果后点击"保存"

### 文档浏览

1. 在文档列表页面可查看所有已上传文档
2. 支持按分类、标签、时间等条件筛选
3. 点击文档标题可查看文档详情
4. 支持文档预览和下载

### 文档搜索

1. 在顶部搜索框输入关键词
2. 系统会在所有文档中进行全文搜索
3. 搜索结果按相关性排序显示

## 分类管理

### 分类模板

1. 点击侧边栏"分类管理"->"模板管理"
2. 可创建、编辑、删除分类模板
3. 模板支持多级分类结构设计

### 分类维护

1. 点击侧边栏"分类管理"->"分类维护"
2. 可查看所有分类及其文档数量
3. 支持手动调整文档分类

## 系统设置

### 用户管理

1. 点击侧边栏"系统管理"->"用户管理"
2. 可创建、编辑、删除系统用户
3. 支持分配不同角色权限

### 系统配置

1. 点击侧边栏"系统管理"->"系统配置"
2. 可配置系统基本参数
3. 修改配置后需重启服务生效

## 高级功能

### 智能分类

系统基于大语言模型自动对文档进行分类：

1. 上传文档后系统自动分析内容
2. 根据预设分类模板进行匹配
3. 提供置信度评分供用户确认

### 信息抽取

系统可自动抽取文档中的关键信息：

1. 文档标题
2. 创建时间
3. 关键词标签
4. 文档摘要

### 自动编号

系统支持按规则自动生成文档编号：

1. 可配置编号规则
2. 支持按分类、时间等维度编号
3. 确保编号唯一性和连续性

## 常见问题

### 文档上传失败

1. 检查文件格式是否支持
2. 确认文件大小未超过限制
3. 查看系统日志了解具体错误信息

### 搜索无结果

1. 确认关键词拼写正确
2. 尝试使用更广泛的搜索词
3. 检查文档是否已正确索引

### 分类不准确

1. 可手动调整文档分类
2. 反馈给系统管理员优化分类模板
3. 联系技术支持改进模型效果
